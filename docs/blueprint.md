# **App Name**: CloudSight Analyzer

## Core Features:

- Traffic Data Aggregation: Collect network traffic data from AWS, Azure, and GCP using their respective monitoring tools.
- Pattern Detection Engine: Identify and classify recurring patterns in network traffic using a rules-based engine, leveraging generative AI to update those rules as a background tool.
- Anomaly Alerting: Generate alerts when unusual traffic patterns are detected, based on a configurable thresholds.
- Visualization Dashboard: Display network traffic patterns in an interactive dashboard, highlighting anomalies and key trends.
- Custom Reporting: Generate custom reports on network traffic, tailored to specific time ranges and metrics.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) to represent cloud infrastructure.
- Background color: Light gray (#D3D3D3), providing a neutral backdrop for data visualization.
- Accent color: Electric violet (#8F00FF) for alerts and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern look.
- Use icons representing network elements, cloud services, and traffic types, in a simple, outlined style.
- Dashboard layout will be based on grid, with expandable panels. Use of white space will give the page balanced composition.
- Subtle animations and transitions on chart updates and user interactions to enhance engagement and provide feedback.
